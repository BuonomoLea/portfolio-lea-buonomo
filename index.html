Projet 4 : Mon guide animal crossing new horizons

Projet 4 : ACNH Guide

Mon ACNH Guide est une application personnelle pour suivre ma progression à 100% dans Animal Crossing: New Horizons.

J’aime aller au bout des choses avec une bonne organisation, donc ce projet m’est très utile, en plus d’être un excellent entraînement.

Principalement codé en HTML, CSS avec Grid, Flexbox et variables, ainsi que JavaScript avec des modules, fetch, async/await.

Il repose sur des composants réutilisables, des fichiers JSON statiques et le stockage local.

Détails du projet 4

Ce projet vise un code léger et modulaire grâce aux Web Components.

J’en ai profité pour approfondir ma maîtrise de fetch.

Améliorations récentes :


> Ajustement des illustrations
> Nettoyage du code CSS
> Amélioration de l’accessibilité clavier

Ce projet m’a permis d’avancer dans mes objectifs qu’ils soient dans mon apprentissage ou dans le jeu.

Actuellement, je crée ma propre bibliothèque UI dédiée à ce type de projet créatif, des illustrations, des abstracts, des blobs et éléments graphiques.
